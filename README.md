# KSMaster
KSMaster is an open source software developed by ASTCOJ-Dev-Team.
The KSMaster GUI version is currently under development.
# Features
- Automaticlly arrange all your students' seats.
- Arranging seats at random.
- Arranging seats by students' exam scores.
# Getting Started
Drag and drop csv files and/or directories into the executable or run:
`ksmaster path...`
`ksmaster path... -o path...`
The csv file should be like that:
```
	stunum,stuname,sturank
	1,Chen Example,1
	...
```
